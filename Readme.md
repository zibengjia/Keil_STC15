**STC15 简易电子琴（项目说明）**

概述

本项目基于 STC15 系列单片机，驱动 LCD1602、DS1302 实时时钟、74HC595 驱动器和蜂鸣器，实现一个带时钟功能的简易电子琴。项目包含手动演奏、自动播放、时钟显示与基本设置菜单。

主要特性

- 支持手动演奏与按键录入
- 自动播放内置曲目（示例若干首）
- LCD1602 菜单显示与曲目信息
- 基本时钟显示及校准功能

硬件要求

- MCU: STC15 系列
- 显示: LCD1602
- 实时时钟: DS1302（可选）
- 扩展移位寄存器: 74HC595
- 输入: 矩阵按键
- 输出: 蜂鸣器/扬声器

软件与开发环境

- 使用 Keil uVision 编译，工程文件：[STC15.uvproj](STC15.uvproj)
- 在本仓库中可使用 VS Code 任务运行构建与烧写（若已配置插件）：运行 `build` / `flash` / `build and flash` 任务

构建与烧录（简要）

1. 用 Keil 打开 [STC15.uvproj](STC15.uvproj) 并编译。
2. 使用你的下载器（如 STC-ISP / 硬件烧录器）将生成的固件烧录到目标板。
3. 在 VS Code 中也可使用任务面板的 `build` / `flash`（如果已配置相应扩展）。

使用说明（快速）

1. 上电后 LCD 显示主菜单（电子琴 / 自动播放 / 时钟设置）。
2. 在电子琴模式，通过按键演奏或切换至自动播放查看内置曲目。
3. 在时钟设置中，可校准日期时间（若接入 DS1302）。
